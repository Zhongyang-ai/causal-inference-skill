# IPW(逆概率加权)原理与适用场景

## 一句话概括

IPW(Inverse Probability Weighting,逆概率加权)用**倾向得分的倒数**给每个样本重新加权,构造一个处理分配与协变量独立的"伪总体(pseudo-population)",从而在观测数据上无偏地估计平均处理效应。

## 适用场景

适合用 IPW 的情况:

- 想估计 **ATE(Average Treatment Effect,总体平均处理效应)**:IPW 用全部样本加权,天然适合估计整体人群的平均效应(也可改权重估 ATT)。
- 数据是**观测数据(observational data)**,处理(treatment)不是随机分配,但混淆变量(confounders)可观测。
- 处理组与对照组在协变量上**重叠良好(good overlap / positivity)**,即两组的倾向得分分布有充分的公共支撑。
- 想用单一模型(倾向模型)同时调整多个混淆变量,而不愿像 PSM 那样丢弃样本。

不适合 / 需谨慎的情况:

- **弱重叠(weak overlap)**:某些协变量区域几乎只有处理组或只有对照组,倾向得分接近 0 或 1,权重会爆炸。
- **极端权重(extreme weights)**:少数样本占据绝大部分权重,点估计被这几个样本主导,方差巨大、不稳定。
- 倾向模型严重误设(misspecified)且无法用双稳健方法补救时。
- 存在**未观测混淆(unmeasured confounding)**:IPW 和所有基于倾向得分的方法一样无能为力。

## 核心原理

**直觉**:在观测数据里,某类样本之所以更容易进入处理组,是因为它们的协变量 $X$ 让倾向得分 $e(x)=P(T=1\mid X=x)$ 偏高。这些样本在处理组里"过度代表"、在对照组里"代表不足"。IPW 的做法是:**给被过度代表的样本降权、给代表不足的样本升权**,用倒数权重把两组都还原成原始总体的样子,从而消除混淆。

**权重公式(ATE)**:

- 处理组样本权重:$w_i = \dfrac{1}{e(x_i)}$
- 对照组样本权重:$w_i = \dfrac{1}{1-e(x_i)}$

加权后,处理分配 $T$ 与协变量 $X$ 独立,加权样本就近似一次随机试验。

**Horvitz-Thompson(HT)vs Hajek(自归一化)**:

- **Horvitz-Thompson 估计量**:直接用权重对结果求和再除以样本量,$\hat{E}[Y(1)]=\frac{1}{n}\sum_i \frac{T_i Y_i}{e(x_i)}$。无偏,但权重之和不等于 $n$ 时方差大。
- **Hajek 估计量(自归一化, self-normalized)**:用每组权重之和做分母,$\hat{E}[Y(1)]=\dfrac{\sum_i T_i Y_i / e(x_i)}{\sum_i T_i / e(x_i)}$。轻微有偏但方差通常小很多,实践中更稳健,是默认推荐。

**Stabilized weights(稳定化权重)降方差**:

把权重改写为

$$w_i=\frac{P(T=T_i)}{P(T=T_i\mid X_i)}$$

即处理组用 $\dfrac{P(T=1)}{e(x_i)}$、对照组用 $\dfrac{P(T=0)}{1-e(x_i)}$。分子的边际概率 $P(T)$ 是常数,不改变目标估计量,但让权重整体围绕 1 波动,**显著降低权重方差和有效样本量损失**。这是边际结构模型(marginal structural model)里的标准做法。

**加权估计 ATE**:

$$\widehat{ATE}=\hat{E}[Y(1)]-\hat{E}[Y(0)]$$

用 Hajek 形式即对处理组、对照组分别算加权均值再相减;等价地可对 $Y\sim T$ 做**加权回归**,回归系数就是 ATE。

## 关键假设

1. **可忽略性 / 无混淆(Ignorability / Unconfoundedness)**:给定可观测协变量 $X$,处理分配与潜在结果独立,$\{Y(0),Y(1)\}\perp T \mid X$。违反后果:存在未观测混淆时,IPW 估计有偏,且无法从数据内部察觉。

2. **Positivity / Overlap(正概率 / 重叠,最关键)**:每个协变量取值下,处理和对照都有正概率,即 $0<e(x)<1$。这是 IPW 区别于 PSM 的命门——权重是倾向得分的**倒数**,一旦 $e(x)\to0$ 或 $\to1$,权重 $\to\infty$。违反后果:极端权重让少数样本主导估计,方差爆炸、点估计漂移,且会在无支撑的区域**静默外推**。这也是必须检查权重分布、做 trimming/truncation、看有效样本量的根本原因。

3. **SUTVA(Stable Unit Treatment Value Assumption)**:个体的潜在结果只取决于自己的处理,不受他人处理影响(无干扰),且处理"版本"一致。违反后果(如网络溢出、一般均衡效应):潜在结果定义本身失效,估计无意义。

4. **倾向模型正确设定(Correct propensity model specification)**:$e(x)$ 的函数形式(变量、交互、非线性)正确。违反后果:错误的倾向得分 → 错误的权重 → 有偏的 ATE,且纯 IPW **没有内部报警**。这正是双稳健方法(AIPW)被推荐的原因。

## 与其它方法的关系

- **与 PSM(倾向得分匹配)**:两者都基于倾向得分 $e(x)$。PSM 给每个处理样本**匹配**相近的对照样本,丢弃匹配不上的样本,默认估 ATT;IPW **不丢样本**,用倒数权重把全样本重新加权,更自然地估 ATE。PSM 直观、对极端样本不敏感但损失样本量;IPW 用满全样本但受极端权重困扰。

- **与 AIPW(Augmented IPW,增广逆概率加权 / 双稳健)**:AIPW = IPW + 结果回归(outcome regression),把加权与回归调整结合。它具有**双稳健性(doubly robust)**:只要倾向模型**或**结果模型有一个设定正确,估计就一致。因此 AIPW 比纯 IPW 更稳健于模型误设,实践中常作为 IPW 的升级版。纯 IPW 仅依赖倾向模型正确。

- **与 DID(Difference-in-Differences,双重差分)**:DID 依赖"平行趋势"假设,用时间维度的前后差分消除不随时间变化的混淆,适合面板/前后对照数据;IPW 依赖"可忽略性 + positivity",用横截面的协变量加权消除可观测混淆,不需要时间维度。两者识别假设不同。

- **与 Uplift Modeling(增益建模)**:Uplift 关注**个体层面(CATE / ITE)**的异质处理效应、用于打分和定向投放;IPW 关注**总体或子总体的平均效应(ATE/ATT)**、用于因果效应的无偏估计与推断。目标粒度不同。

## 常见坑

- **极端权重不 trimming/truncation**:倾向得分接近 0 或 1 时权重飞到几十上百,几个样本就主导整个估计。务必检查最大权重、最大/均值比、尾部权重占比,必要时按分位(如 1%/99%)截断或修剪。

- **不看有效样本量(ESS, Effective Sample Size)**:$ESS=\dfrac{(\sum_i w_i)^2}{\sum_i w_i^2}$。权重越不均匀,ESS 相对名义样本量掉得越多。ESS 大幅下降意味着估计实际上只靠少数样本撑着,置信区间会很宽。

- **不检验加权后平衡**:加权的目的就是让协变量在两组间平衡。必须算**加权 SMD(weighted standardized mean difference)**,目标 $|SMD|<0.1$。倾向模型预测得"准"不等于平衡达标——平衡才是验收标准。

- **倾向模型误设**:漏掉重要混淆、缺少交互项或非线性项,会让权重错误从而 ATE 有偏。纯 IPW 对此无内部报警;条件允许时改用双稳健的 AIPW。

- **权重未稳定化(unstabilized)**:不做 stabilization 的权重方差天然更大、ESS 更低。同样的点估计,稳定化权重几乎总是更稳。
