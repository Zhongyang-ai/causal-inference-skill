# AIPW(增广逆概率加权 / 双稳健)原理与适用场景

## 一句话概括

AIPW(Augmented Inverse Probability Weighting,增广逆概率加权)又称双稳健估计(Doubly Robust, DR),它把**结果回归模型**和**倾向得分加权(IPW)** 结合成一个估计量:只要**结果模型**和**倾向模型**这两者之中**至少有一个设定正确**,对平均处理效应(ATE)的估计就是一致的(consistent)——给了你两次机会而不是一次。

## 适用场景

适合用 AIPW 的情况:

- 想要一个**对模型误设更稳健**的 ATE 估计,不愿把全部赌注押在单一模型(纯回归或纯 IPW)上。
- 你**既能建结果模型,也能建倾向模型**,愿意同时维护两个 nuisance(扰动)模型。
- 希望用**机器学习模型**(梯度提升、随机森林等)来拟合结果或倾向,同时还想要有效、有理论保证的推断——AIPW + cross-fitting(即 double machine learning, DML)正是为此设计。
- 样本量中等及以上,倾向得分**重叠(overlap)良好**,positivity 大致满足。

不适合 / 需谨慎:

- **两个模型都没把握**:双稳健的前提是"至少一个正确",两个都错时 AIPW 依旧有偏,并不会变魔术。
- **样本太小**:增广项含 `1/e` 和 `1/(1-e)`,小样本下方差大、不稳定;cross-fitting 进一步压缩每折训练量。
- **重叠很差 / positivity 严重违反**:极端倾向得分会让权重爆炸,方差巨大,任何加权类方法都不可靠。
- 想估计**个体层面**异质效应(那是 uplift / CATE 的范畴),或要控制**不可观测**混淆(那需要 DID、IV 等)。

## 核心原理

**双稳健思想**:纯回归(outcome regression)依赖结果模型 `mu_t(x)=E[Y|T=t,X=x]` 正确;纯 IPW 依赖倾向模型 `e(x)=P(T=1|X=x)` 正确。AIPW 把二者拼起来,使得**任一模型正确即可一致**。

对 ATE 的 AIPW 估计量:

$$
\hat\tau_{AIPW}=\frac{1}{n}\sum_{i=1}^{n}\Big[\hat\mu_1(x_i)-\hat\mu_0(x_i)+\frac{T_i\,(Y_i-\hat\mu_1(x_i))}{\hat e(x_i)}-\frac{(1-T_i)\,(Y_i-\hat\mu_0(x_i))}{1-\hat e(x_i)}\Big]
$$

逐项理解:

- `mu_1(x_i) - mu_0(x_i)`:这是**纯结果回归**给出的个体效应估计,对所有人取平均就是回归法的 ATE。
- 后两项是**"增广项"(augmentation)**,本质是对 IPW 的**残差修正**:它只用每个单位**实际观测到的那一臂**的残差 `Y - mu_t(x)`,再按逆倾向权重放大。
  - 若结果模型正确,残差期望为 0,增广项期望消失,估计量退化为正确的回归估计;
  - 若倾向模型正确,增广项恰好抵消回归模型的偏差,把估计拉回真值。
  - 这两条互补,正是"双稳健"的来源。
- 也可把它读成"**IPW 估计 + 用结果模型做的方差缩减修正**":相比纯 IPW,增广项扣掉了可由 `X` 解释的部分,降低方差。

**Cross-fitting 与 DML 简述**:当 `mu` 和 `e` 用灵活 ML 拟合时,若在同一批数据上既训练又评估,模型会过拟合,使 `\hat\tau` 产生过拟合偏误。Cross-fitting(样本分裂)把数据分 K 折,在**其余折**上训练 nuisance 模型,只在**留出折**上预测,从而让 nuisance 误差与效应估计解耦。这就是 **double machine learning (DML)** 的核心做法,也与 **TMLE**(目标最大似然估计)思路相邻。

**效率(efficiency)**:当两个模型都一致时,AIPW 的影响函数(influence function)正是该问题的**高效影响函数(efficient influence function)**,其方差达到**半参数有效边界(semiparametric efficiency bound)**——即在所有正则估计量中渐近方差最小。基于影响函数的方差估计(逐单位得分的样本方差除以 n)给出渐近有效的标准误与置信区间。

## 关键假设

与一般可识别条件相同的三条:

1. **可忽略性 / 无未观测混淆(ignorability / unconfoundedness)**:给定观测协变量 `X`,处理分配与潜在结果独立。AIPW 同样**只能处理可观测混淆**。
2. **正性 / 重叠(positivity / overlap)**:`0 < e(x) < 1`,每个协变量取值下都有可能进入任一臂,否则权重发散。
3. **SUTVA**:个体间无干扰,处理版本一致。

AIPW 特有的优势条件:

4. **至少一个模型正确(double robustness)**:结果模型 `mu` 与倾向模型 `e` 中**至少一个设定正确**即可保证一致性。
   - 相比**纯 IPW**:纯 IPW 把全部希望压在 `e` 正确上;AIPW 多了一层结果模型保险。
   - 相比**纯结果回归**:纯回归把全部希望压在 `mu` 正确上;AIPW 多了一层倾向加权保险。
   - 注意:双稳健只对**一致性(consistency / 无偏)**成立;要得到**有效推断与最优方差**,通常还需要两个模型都收敛得足够快,并配合 cross-fitting。

## 与其它方法的关系

- **vs 纯 IPW(纯加权)与纯 outcome regression(纯回归)**:AIPW 是二者的**融合升级版**。纯 IPW 只用倾向、纯回归只用结果模型,各押一注;AIPW 同时使用两个模型,任一正确即一致,因此比二者单独更稳健,通常方差也更小。
- **vs PSM(倾向得分匹配)**:PSM 通过匹配构造可比子样本,**会丢弃未匹配样本**、依赖匹配质量与 caliper 设定,且匹配后方差/推断较棘手;AIPW 不丢样本、对全样本加权并融合结果模型,推断有影响函数支撑。两者都只处理**可观测混淆**。
- **vs DID(双重差分)**:DID 用**面板/前后对比**与平行趋势假设来**消除不随时间变化的不可观测混淆**;AIPW 是横截面方法,**不能**控制不可观测混淆。数据结构与假设不同。
- **vs Uplift / CATE(个体处理效应)**:Uplift 建模目标是**异质的、个体层面**的处理效应 `tau(x)`,用于人群定向;AIPW(本文口径)估计的是**总体平均效应 ATE/ATT**。AIPW 的双稳健得分也可作为 CATE 元学习器(如 DR-learner)的伪结局,但默认目标是平均效应。

(以上口径与本仓库 PSM / IPW / DID / Uplift 文档保持一致:可观测 vs 不可观测混淆、平均 vs 个体效应、是否丢样本。)

## 常见坑

- **不做 cross-fitting 导致过拟合偏误**:用灵活 ML 又在同批数据训练+评估,`\hat\tau` 会被过拟合污染、置信区间失真。务必样本分裂、留出折预测。
- **极端倾向得分**:`e(x)` 贴近 0 或 1 时,`1/e`、`1/(1-e)` 爆炸,即便点估计无偏,方差也会失控。需检查重叠、做 trimming / truncation。
- **误以为双稳健 = 无敌**:双稳健是"两次机会",**不是两个模型都错还能对**。两个都误设时 AIPW 照样有偏。
- **两个模型同源误设(common-source misspecification)**:若结果模型与倾向模型**遗漏了同一个混淆变量**或共用了**同样错误的函数形式**,则"至少一个正确"的前提被破坏,双稳健失效。
- **把弱重叠当成模型问题**:AIPW 与 IPW、回归三者结果差异巨大,往往是 overlap / positivity 太差所致,而非单纯调模型能解决。
