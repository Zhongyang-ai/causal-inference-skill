# DID(双重差分)原理与适用场景

## 一句话概括

DID(Difference-in-Differences,双重差分)通过比较「处理组前后变化」与「对照组前后变化」的**差异**,在消除共同时间趋势和组间固定差异的前提下,估计某项干预(政策/产品上线)对处理组的因果效应(ATT)。

## 适用场景

**适合用 DID 的情况:**
- 政策评估、产品功能上线、运营活动等存在明确「干预时点」的自然实验。
- 拥有**前后两期(pre/post)+ 处理组(treated)与对照组(control)**的数据。
- 数据为**面板数据(panel,同一批个体被重复观测)**或**重复横截面(repeated cross-section,同样的组、同样的期,但个体不同)**。
- 存在**不随时间变化的不可观测混淆**(如地区禀赋、用户固有偏好),无法靠观测变量校正,但可被组间差分吸收。

**不适合的情况:**
- 没有合适的对照组(所有个体同时被干预,即全局上线无对照)。
- 处理组与对照组**事前趋势明显不平行**,或干预正是由趋势差异触发(如「下滑的地区才上政策」)。
- 对照组会被干预**间接影响(存在溢出/外溢效应)**,违反 SUTVA。
- 干预时点附近存在只冲击一组的其他事件(时变混淆),无法与处理效应区分。

## 核心原理

DID 的核心思想是「**差分再差分**」:

- 第一重差分(组内前后差)消除每组**不随时间变化的固定差异**(组固定效应)。
- 第二重差分(处理组之差 − 对照组之差)消除**两组共同的时间趋势**(时间固定效应)。
- 剩下的就是干预带来的**增量效应**。

**2x2 DID 公式(两组两期):**

$$\text{DID}=(\bar Y^{T}_{post}-\bar Y^{T}_{pre})-(\bar Y^{C}_{post}-\bar Y^{C}_{pre})$$

其中 $T$ 为处理组,$C$ 为对照组。处理组前后差里既包含时间趋势又包含处理效应;对照组前后差只包含时间趋势;两者相减即得到净处理效应。

**回归形式(与 2x2 完全等价):**

$$Y=\beta_0+\beta_1\, Treat+\beta_2\, Post+\beta_3\,(Treat\times Post)+\epsilon$$

- $\beta_0$:对照组 pre 期均值(基准)。
- $\beta_1$:处理组与对照组的固定水平差(pre 期)。
- $\beta_2$:对照组从 pre 到 post 的共同时间趋势。
- $\beta_3$:**交互项系数,即 DID 估计量,在平行趋势假设下等于 ATT**。

回归形式的优势:可加协变量、可直接得到标准误与置信区间、可聚类。

**TWFE 与多期推广:**

当有多个个体、多个时期时,用**双向固定效应(Two-Way Fixed Effects, TWFE)**:

$$Y_{it}=\alpha_i+\lambda_t+\beta\, D_{it}+\epsilon_{it}$$

其中 $\alpha_i$ 为个体固定效应、$\lambda_t$ 为时间固定效应、$D_{it}$ 为「个体 $i$ 在时期 $t$ 是否处于被处理状态」的指示变量,$\beta$ 即处理效应。两组两期时 TWFE 退化为上面的 2x2 回归。

**Event study(事件研究法):**

把处理与「相对干预时点的各期(leads/lags)」交互,省略某一 pre 期作为基准:

$$Y_{it}=\alpha_i+\lambda_t+\sum_{k\neq -1}\beta_k\, \mathbf{1}\{t-E_i=k\}\cdot Treat_i+\epsilon_{it}$$

- 干预**前(leads)的系数应接近 0** → 佐证平行趋势(无预期效应)。
- 干预**后(lags)的系数**刻画处理效应随时间的动态演化(立刻见效 / 逐步累积 / 衰减)。

## 关键假设

1. **平行趋势(parallel trends)—— 最核心、最关键。**
   假设:若没有干预,处理组与对照组的结果会沿**相同的趋势**变化。
   违反后果:DID 把本就存在的趋势差异误当成处理效应,估计有偏。
   说明:该假设**不可直接检验**(反事实不可观测),只能用多期 pre 数据做 pre-trend 检验/event study 佐证,或用 placebo 检验旁证;通过 pre-trend 检验是**必要而非充分**条件。

2. **无预期效应(no anticipation)。**
   假设:个体在干预正式生效前不会因预期而提前改变行为。
   违反后果:效应「提前泄漏」到 pre 期,event study 的 lead 系数非零,ATT 被低估或时点错配。

3. **SUTVA / 无溢出(no spillover / no interference)。**
   假设:一个个体是否受处理只影响自己,对照组不受处理组的间接影响。
   违反后果:对照组被「污染」(如处理组的政策外溢到邻近对照地区),两组之差被压缩或夸大,估计有偏。

4. **稳定组成(stable composition)。**
   假设:各组在前后期由「可比的同质单位」构成;面板中个体不随干预选择性进出。
   违反后果:**组成变化(composition change)**——尤其在重复横截面中,group 内部个体结构在前后期发生系统性变化,会把结构变化误判为处理效应(选择性进入/退出偏误)。

(此外还隐含:干预时点附近无只冲击单组的其他时变混淆冲击;否则无法与处理效应区分。)

## 与其它方法的关系

- **与 PSM / IPW / AIPW(横截面方法)对比:**
  PSM、IPW、AIPW 是**横截面**方法,核心是用倾向得分**校正可观测的混淆变量**(基于「可忽略性 / 无未观测混淆」假设),只能调整你**观测到**的混淆。
  DID 则需要**面板 / 前后两期**数据,通过组内差分,能额外**吸收掉不随时间变化的不可观测混淆**(time-invariant unobserved confounders)——这是横截面方法做不到的。代价是 DID 依赖**平行趋势**假设,且无法处理**时变**的不可观测混淆。
  口径一致性:三类方法估计的都是干预对结果的因果效应,常以 **ATT** 为目标估计量;区别在识别假设(可观测混淆 vs 不变的不可观测混淆 + 平行趋势)与所需数据结构(横截面 vs 面板/两期)。实务中可结合:对面板做 DID,同时用 PSM/IPW 提升处理组与对照组的事前可比性(matched / weighted DID)。

- **与 Uplift(增益建模)区别:**
  Uplift 关注**个体层面的异质处理效应**(CATE,谁对干预最敏感),服务于精准投放/定向决策;DID 关注的是**组层面的平均处理效应(ATT)**,服务于「这项干预整体有没有效、效应多大」的评估问题。二者目标不同:DID 回答「有多大效应」,Uplift 回答「对谁更有效」。

## 常见坑

1. **不检验 pre-trend 就直接做 DID。**
   不画 event study / 不看事前趋势,平行趋势是否成立完全无据。哪怕只有两期,也应尽量找历史数据或 placebo 旁证。

2. **staggered adoption(交错/分批接受处理)下直接套朴素 TWFE。**
   当不同个体在不同时间被处理且效应异质时,朴素 TWFE 会用**已处理单位充当对照**(forbidden comparison),导致估计有偏甚至**符号反转**。应使用现代估计量(Callaway-Sant'Anna、Sun-Abraham、de Chaisemartin-D'Haultfoeuille),或先做 Goodman-Bacon 分解诊断。

3. **对照组被间接影响(溢出 / 污染对照)。**
   处理组的干预外溢到对照组(地理相邻、用户互动、市场竞争),违反 SUTVA,使两组之差被稀释或扭曲。选对照组时要尽量隔离溢出。

4. **把水平差异当成处理效应。**
   两组事前本就存在水平差(level difference)是正常的(被 $\beta_1$ / 个体固定效应吸收),**只有干预后的「差异化变化」($\beta_3$)才是效应**。直接比较 post 期两组的绝对水平,或只看处理组前后差,都会把固定差异/共同趋势误算进效应。
