# PSM(倾向得分匹配)原理与适用场景

## 一句话概括

PSM 通过估计每个个体"接受处理(treatment)的概率"(倾向得分),把混淆变量压缩成一个分数,再让处理组和对照组在这个分数上配对,从而在**观测数据**中近似一次随机实验,估计处理对结果的因果效应。

## 适用场景

**适合用 PSM 的业务问题:**

- 营销增量评估:对触达过某活动(发券/推送/广告)的人群 vs 未触达人群,估计活动带来的增量转化、增量 GMV。
- 渠道/功能投放效果:是否在某渠道投放、是否开通某功能(如开通自动续费、参加会员体系)对留存/付费的影响。
- 运营策略对照:某策略只发给了部分用户(非随机,而是按规则或人工挑选),想估计该策略的真实效果。
- 任何"有明确二元处理 + 有结果指标 + 有一批可观测的、处理前的特征"的横截面观测研究。

**不适合用 PSM 的场景:**

- **已经有随机实验(A/B test)**:直接对比即可,不需要 PSM。
- **混淆变量大多不可观测**:PSM 只能校正你**测量得到**的混淆,测不到的(如用户内在购买意愿)无能为力,此时考虑 DID、IV 等。
- **处理几乎确定**:某类人 100% 会被处理、另一类 100% 不会(共同支撑严重缺失),没有可比对象,PSM 失效。
- **关心个体层面的异质性效应**(对谁有效、对谁无效):应使用 Uplift / CATE 类方法,PSM 给的是平均效应。
- **处理不是清晰的二元干预**(连续剂量、多值处理):标准 PSM 不直接适用,需要广义倾向得分等扩展。

## 核心原理

### 倾向得分的定义

对每个个体,给定其处理前协变量 $X$,定义**倾向得分**为它被分到处理组的条件概率:

$$e(X) = P(T = 1 \mid X)$$

其中 $T \in \{0, 1\}$ 表示是否接受处理。实践中常用 logistic regression 把 $X$ 映射到 $e(X)$。

### 为什么能降维并平衡混淆

混淆变量 $X$ 可能有几十维,直接在每一维上精确配对几乎不可能(维度灾难)。Rosenbaum 与 Rubin(1983)证明了一个关键结论:**倾向得分是一个平衡分(balancing score)**——在给定 $e(X)$ 的条件下,处理组与对照组的协变量分布相同:

$$X \perp T \mid e(X)$$

也就是说,只要两个个体的倾向得分相同,即便他们的原始 $X$ 不完全一样,在统计意义上他们的混淆结构是可比的。于是我们把"在高维 $X$ 上平衡"这个难题,简化为"在一维标量 $e(X)$ 上平衡"。

### 匹配、共同支撑与平衡检验

- **匹配(matching)**:为每个处理组个体,找一个(或多个)倾向得分最接近的对照组个体配成对。常用 1:1 最近邻匹配,并加 **caliper(卡钳)** 限制——只有当得分距离小于阈值(常用 $0.2 \times \text{SD}(\text{logit}(e(X)))$)时才允许配对,避免"硬凑"出质量很差的对子。匹配通常在 logit 尺度 $\text{logit}(e) = \ln\frac{e}{1-e}$ 上做,因为它在 0/1 附近被拉开,距离更合理。
- **共同支撑 / 重叠(common support / overlap)**:处理组和对照组的倾向得分分布要有足够重叠区间。落在对照组得分范围之外的处理组个体,根本找不到可比对象,需要丢弃或标记。
- **平衡检验 SMD(标准化均差)**:匹配的目标**不是**预测得准,而是匹配后两组协变量分布要接近。对每个协变量计算标准化均差:

$$\text{SMD} = \frac{\bar{X}_{\text{treat}} - \bar{X}_{\text{control}}}{\sqrt{(s^2_{\text{treat}} + s^2_{\text{control}})/2}}$$

匹配后每个协变量的 $|\text{SMD}|$ 一般要求小于 0.1(严格场景 0.05)。SMD 的好处是无量纲、不受样本量影响,可直接横向比较。

### ATT vs ATE

- **ATT(Average Treatment effect on the Treated)**:处理对"实际被处理的那群人"的平均效应。1:1 最近邻匹配天然估计的是 ATT——因为我们以处理组为锚,为他们找对照。
- **ATE(Average Treatment Effect)**:处理对"全体人群"(假设所有人都被处理 vs 都不被处理)的平均效应。
- 业务上"我们这次活动对参与者带来多少增量"问的是 ATT;"如果把活动推给所有人会怎样"问的是 ATE。两者在效应异质时数值不同,报告时务必说清估计的是哪一个。

## 关键假设

PSM 给出的是因果而非相关的结论,前提是下面三条假设成立。任何一条被破坏,结论都可能失真。

1. **可忽略性 / 无混淆(Unconfoundedness / CIA, Conditional Independence Assumption)**:
   在控制了观测协变量 $X$ 后,处理分配与潜在结果独立:$(Y_0, Y_1) \perp T \mid X$。
   通俗说:所有同时影响"是否被处理"和"结果"的因素都已被测量并放进了模型。
   **违反后果**:存在未观测混淆(如用户固有意愿)时,估计是有偏的,可能把选择效应误当成处理效应。这是 PSM 最致命也最难验证的假设——数据本身无法证明它成立,只能靠领域知识论证 + 敏感性分析。

2. **共同支撑 / 正性(Overlap / Positivity)**:
   在协变量任意取值处,被处理和不被处理的概率都严格介于 0 和 1 之间:$0 < e(X) < 1$。
   通俗说:每一类人里,处理组和对照组都得有人,才能比较。
   **违反后果**:某些个体没有可比对象,匹配被迫外推或丢弃大量样本,方差暴涨、估计不稳,结论只能局限在有重叠的子人群上。

3. **SUTVA(Stable Unit Treatment Value Assumption)**:
   个体的潜在结果只取决于自己是否被处理,不受他人处理状态影响(无干扰/溢出效应),且处理只有单一版本。
   **违反后果**:存在网络溢出(如对照组用户被处理组用户安利)或处理本身有多种强度时,潜在结果定义被破坏,效应估计失去意义。

## 与其它方法的关系

| 方法 | 校正什么 | 适用数据结构 | 关注的量 |
|------|----------|--------------|----------|
| **PSM** | 仅可观测混淆 | 横截面观测数据 | 平均效应(ATT 为主,也可 ATE) |
| **DID(双重差分)** | 可吸收不随时间变的**不可观测**混淆 | 前后至少两期 + 处理组与对照组 | 平均效应(ATT) |
| **IPW(逆概率加权)** | 仅可观测混淆(用倾向得分做权重而非配对) | 横截面观测数据 | 平均效应(自然估 ATE) |
| **AIPW(增广 IPW / 双稳健)** | 仅可观测混淆,但结合结果模型 | 横截面观测数据 | 平均效应(ATE/ATT),**双稳健** |
| **Uplift / CATE** | 可观测混淆 | 实验或观测数据 | **个体异质性**效应 |

选择口径:

- **PSM**:只有横截面、只有可观测混淆、想要直观可解释的匹配队列时用。本 skill 的默认方法。
- **DID**:当你有处理前后两期数据 + 一个平行的对照组,且担心存在不随时间变化的不可观测混淆(如品牌固有差异)时,优先 DID。它靠"前后差的差"把固定的不可观测因素消掉,这是 PSM 做不到的。
- **IPW / AIPW**:同样只校正可观测混淆,但用加权代替配对,能用上全部样本(不丢弃)。**AIPW 双稳健**——倾向模型或结果模型任一个设对,估计就一致,稳健性优于纯 PSM 或纯 IPW。重叠尚可、想估 ATE 时优先考虑。
- **Uplift**:当业务问题是"该把活动发给哪些人(对谁增量最大)"而不是"平均增量多少"时,用 Uplift 建模 CATE,而非 PSM。

## 常见坑

- **匹配后不做平衡检验**:倾向模型预测得再准也不代表协变量平衡了;必须看匹配后 SMD,而不是看分类 AUC。
- **把 treatment 之后的变量放进倾向模型**:加入处理之后才发生的变量(mediator 中介、collider 对撞、结果衍生特征)会引入偏倚甚至制造假因果。倾向模型只能用**处理前**的协变量。
- **弱重叠却硬估计**:共同支撑很差时仍强行匹配/外推,得到的"效应"主要来自外推假设而非数据,极不可靠;此时应如实说明数据不足以支撑该因果问题。
- **把 ATT 当 ATE 汇报**:1:1 匹配估的是 ATT,不要包装成全人群结论。
- **caliper 设得过松或不设**:会配出大量低质量对子,看似样本多实则引入偏倚;过紧又会丢太多样本、方差变大,需要权衡并报告丢弃比例。
- **忽略匹配带来的不确定性**:匹配是一个估计步骤,标准误应通过 bootstrap 或匹配感知的方差公式得到,而不是直接套用普通 t 检验。
- **只看点估计不看置信区间**:增量效应若置信区间跨过 0,不能宣称"有显著增量"。
