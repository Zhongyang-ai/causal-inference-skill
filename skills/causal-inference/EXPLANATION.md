# 因果推断方法选择指南（Router）

这是因果推断技能集的**总入口**。当你有一个"某个动作 / 投放 / 政策到底有没有效果、效果多大"的问题，但还没想好用哪种方法时，先看这份指南。它会根据你的**数据形态**和**业务目标**，帮你选出最合适的方法，再跳转到对应的子技能。

## 一句话概括

先问两件事——**你要的是"平均效应"还是"对谁有效"**，以及**你的数据是"一张横截面快照"还是"有前后两期+对照组"**——这两个分叉就能定下用哪种方法。

## 五种方法速览

| 方法 | 子技能 | 估计目标 | 核心数据要求 | 一句话定位 |
| --- | --- | --- | --- | --- |
| PSM 倾向得分匹配 | `psm-causal-inference` | ATT（也可 ATE） | 横截面观测数据 + 可观测混淆 | 给处理组找"双胞胎"对照，匹配后比差异 |
| IPW 逆概率加权 | `ipw-causal-inference` | ATE / ATT | 横截面观测数据 + 重叠良好 | 用倾向得分倒数加权，构造伪总体 |
| AIPW 双稳健 | `aipw-causal-inference` | ATE / ATT | 观测数据 + 想抗模型误设 | IPW + 结果回归，二者有一个对就行 |
| DID 双重差分 | `did-causal-inference` | ATT | 面板/重复截面，有前后期 + 对照组 | "处理组前后差 − 对照组前后差" |
| Uplift 增益建模 | `uplift-modeling` | 个体/子群 CATE | 处理+结果，最好随机实验 | 估个体异质效应，回答"该对谁投放" |

## 决策逻辑（按顺序走，第一个命中就是首选）

### 第一分叉：你要"平均效应"还是"对谁有效"？

- **想知道"该对谁投放 / 谁是被打动者(persuadables) / 个体回报"** → 用 **Uplift Modeling**。它估的是个体/子群异质效应(CATE)，不是一个平均数。
  - 注意：Uplift 最适合**随机实验/A-B test** 数据；如果是观测数据，要**先用 PSM/IPW/AIPW 去混淆**，再建 CATE，否则每个子群的效应都是有偏的。
- **想知道"这个动作平均带来多大效果"(ATE/ATT)** → 继续看第二分叉。

### 第二分叉：你的数据有没有"前后两期 + 对照组"？

- **有**：某政策/功能在某个时间点上线，你能观测到处理组和一个可比对照组在**上线前和上线后**的结果（面板或重复截面）→ 用 **DID**。
  - DID 最大的好处：它能消掉**不随时间变化的不可观测混淆**——这是 PSM/IPW/AIPW 都做不到的。
  - 代价：依赖**平行趋势(parallel trends)**假设，务必检查事前趋势(pre-trend)。
- **没有**（只有一张横截面快照、处理非随机分配、想要平均效应）→ 这是倾向得分家族问题，看第三分叉在三者中挑。

### 第三分叉：横截面观测数据，在 PSM / IPW / AIPW 中怎么选？

- **想要可解释的匹配队列、要向业务方展示"可比的处理 vs 未处理个体"、关注 ATT** → **PSM**。直观，但会丢弃无法匹配的样本。
- **想要全样本估计总体 ATE、且重叠(overlap)良好** → **IPW**。用全部样本加权，但要警惕极端权重 / positivity。
- **不确定哪个模型设得对、或想用 ML 模型、想要更稳健** → **AIPW（双稳健）**。只要倾向模型和结果模型**有一个**正确就一致；建议 cross-fitting。**当你能同时建两个模型时，AIPW 是三者里最稳妥的默认选择。**

### 特殊情况：干净的随机实验(A/B test)

如果是干净随机分配且只要平均效应，**直接做组间均值差**就已经无偏，不一定需要因果推断机器。IPW/AIPW 只在随机化不完美/有流失时用来校正，或当问题变成"对谁投放"时转 Uplift。

## 推荐时的输出规范

选完方法，要向用户讲清楚：
1. 因果问题：treatment、outcome、单位、人群、时间窗、这个答案要支撑什么决策。
2. 估计目标(ATE/ATT/CATE)以及为什么匹配这个决策。
3. 数据诊断：分配机制(随机 vs 观测)、时间结构、有无对照组、样本量、重叠。
4. 推荐的方法 + 它依赖的关键假设 + 它**修不了**的那种偏误。
5. 尽量给一个稳健性/三角验证结果（如 PSM 与 IPW 互相印证）。
6. 残余风险声明（不可观测混淆、弱重叠、平行趋势不可检验等）。

## 常见坑

- **没有任何方法能拯救坏设计**：既无随机化、又无前后期、又无重叠，就如实说"这个数据回答不了这个因果问题"。
- **估计目标要匹配决策，而不是图方便**：总体 ATE 和"该对谁投放"是两个不同的问题。
- **DID 的平行趋势 与 倾向家族的可忽略性 是两套不同的逃生舱**：DID 能吸收不随时间变的不可观测混淆但需要面板数据；PSM/IPW/AIPW 只校正可观测混淆但能用单期快照。
- **观测数据上不要直接建 Uplift CATE**：先去混淆，否则子群效应有偏。
- **倾向/结果模型预测准不是目标**：平衡(PSM/IPW)、双稳健(AIPW)、排序质量/Qini(Uplift)才是。
